# MobiQuest

Мобильное приложение, которое помогает Flutter-разработчикам изучать и закреплять знания с помощью коротких тематических тестов и краткой теории.

## Функциональность

- Небольшие тесты по различным темам Flutter и мобильной разработки
- Краткая теория по каждой теме
- Добавление новых тем без необходимости обновлять приложение
- Профиль пользователя с накопленным опытом


## Технологии

- **Flutter** — фреймворк для UI
- **BLoC** — управление состоянием
- **Hive** — локальное хранилище
- **Dio** — работа с сетью


## Принципы разработки

- Clean Architecture
- Принципы SOLID
- Repository Pattern
- Dependency Injection (get_it)
- Адаптивный UI — все размеры рассчитываются от ширины экрана
